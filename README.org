|------------+---------|
*** Setup
#+BEGIN_SRC shell
git clone https://github.com/anhsirk0/tendrils-frontend.git --depth=0
cd tendrils-frontend
pnpm install
pnpm run dev
#+END_SRC
[[https://github.com/anhsirk0/tendrils-backend][tendrils-backend]] is also needed to be running on port 8000
